## [CVPR 2026] FEAT: Fashion Editing and Try-On from Any Design

**FEAT** is a method for flexible fashion editing and virtual try-on from arbitrary designs.

Notably, this work was selected as a **Highlight paper at CVPR 2026**, one of the top conferences in computer vision.

### 🔍 Key Features

* 👗 Edit clothing designs directly from input references
* 🔄 Seamless virtual try-on with diverse styles
* 🎨 High-fidelity texture and pattern transfer
* 🤖 Applicable to various fashion AI tasks

### 🏆 Achievement

* Selected as a **CVPR 2026 Highlight** (top-tier recognition)
